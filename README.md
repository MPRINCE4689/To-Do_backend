
# To-Do List Backend

This is the backend for a To-Do List application built with:
- Node.js
- Express.js
- MongoDB (Local via Compass)
- Mongoose (ODM)

## Features
✅ Create Task  
✅ Read All Tasks  
✅ Update Task  
✅ Delete Task  
✅ MongoDB Compass storage

## Installation
```bash
npm install
npx nodemon server.js
